## 🚀 Sobre el Proyecto

Este repositorio contiene el código fuente de la página web estática de **ApMobiTech Developer**. Diseñado de forma limpia, rápida y sin dependencias externas, su propósito principal es doble:

1. **Portafolio Profesional:** Una carta de presentación digital para exhibir aplicaciones móviles, diseño de interfaces UI/UX y servicios de desarrollo e integración de APIs.
2. **Cumplimiento Legal (Google Play):** Alojar la [Política de Privacidad](privacy.html) oficial requerida por las normativas de transparencia y seguridad de datos de Google Play Console para la publicación de aplicaciones.

## 🛠️ Tecnologías y Enfoque

Para garantizar la máxima velocidad de carga, mantenibilidad cero y evitar problemas de enrutamiento con los bots de revisión de Google, el proyecto se ha construido como un **sitio web estático puro (Multi-página)**:

* **Estructura:** HTML5 semántico.
* **Estilos:** CSS3 nativo con uso de variables (*Custom Properties*) para mantener la paleta de colores de la marca (Azul, Magenta, Naranja).
* **Diseño:** 100% Responsivo adaptado a dispositivos móviles, tablets y escritorio.
* **Hosting:** Despliegue continuo a través de la infraestructura global de GitHub Pages.

## 📂 Estructura de Archivos

La arquitectura plana del repositorio consta de los archivos estrictamente necesarios:

```text
/
├── index.html       # Página principal (Hero, Servicios, Contacto)
├── privacy.html     # Documento legal exigido por las tiendas de apps
├── styles.css       # Hoja de estilos global unificada
├── logo.jpg         # Identidad visual corporativa y Favicon
└── banner.jpg       # Imagen de cabecera para la sección Hero
````

## 🌐 Configuración de GitHub Pages
Este sitio está diseñado para funcionar "Out of the box" con GitHub Pages.
  1. Sube los 5 archivos a la rama principal (main).
  2. Dirígete a Settings > Pages en este repositorio.
  3. En la sección Build and deployment, selecciona la rama main y la carpeta / (root).
  4. Guarda los cambios. Tu sitio estará disponible en unos minutos.
